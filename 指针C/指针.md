# 内存单元
真正代表A的是第一个内存单元

# 指针变量
存地址的变量

# 指针使用
*p 指针的解引用, 或者说间接引用
将p的内容取出, 然后当成地址看待, 找到内存空间(这一点很重要)
左值: 存数据到空间中, 将数据存储到刚才找到的内存空间中, 直接改地址
右值: 取出空间中的内容

# 指针大小
指针大小和类型无关
只和操作系统相关
# 野指针
1. 没有有效地址空间
int *p; // 随机声明,并分配了一个随机空间到p
*p = 1000;
2. 地址空间不能访问
int *p = 100; // 0-255 是操作系统用的地址, 禁止访问
*p = 200;
# 泛型指针
void *p; // 万能指针
使用的话需要做强制转化
*(int *)p;

# 指针数组
1. 数组名是地址常量
a[0] 相当于 *(a + 0)

# const 指针
常用来限制函数的形参
